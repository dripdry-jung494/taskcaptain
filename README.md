## 🆘 Troubleshooting

- **The program won’t start:**  
  Restart your PC, then try launching taskcaptain again. Make sure your Windows version is supported.

- **Installation fails or stops:**  
  Check your disk space and close any running programs. Try running the installer as administrator (right-click the file, choose “Run as administrator”).

- **Tasks don’t save:**  
  Ensure you have write permission in the app folder. Try exporting your tasks, then reinstall taskcaptain.

- **Notifications not showing:**  
  Check your Windows notification settings to allow alerts from taskcaptain.
